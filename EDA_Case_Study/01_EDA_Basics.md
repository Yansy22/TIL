# EDA (탐색적 데이터 분석) 기초

## 1. EDA의 목적
- 데이터를 본격적으로 분석하기 전에, 데이터를 다양한 각도에서 관찰하고 이해하는 과정.
- 데이터의 구조, 패턴, 이상치, 변수 간 관계를 파악하는 것이 목표.

## 2. EDA의 4가지 핵심 단계
1.  **데이터 정제 (Cleaning):** 결측치, 이상치 처리
2.  **데이터 요약 (Summarizing):** 기초 통계량 확인 (평균, 중앙값 등)
3.  **데이터 시각화 (Visualization):** 히스토그램, 산점도 등으로 패턴 파악
4.  **가설 수립:** 분석을 통해 얻은 인사이트로 가설 설정
